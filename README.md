# C Programming Practice

This repository contains topic-wise C programming examples created during academic practice.

## Topics Covered
- Basics & Control Statements
- Loops & Arrays
- Functions & Pointers
- Structures & Dynamic Memory Allocation
- File Handling (using relative paths)

## Notes
- Compiled files (.exe) are ignored using `.gitignore`
- Text files used for file handling programs are stored in `text-files/`

This repository is intended for learning and practice purposes.
